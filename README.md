## Live Demo

Official site: [https://ythreads.vercel.app/](https://ythreads.vercel.app/)

### Features

- Create threads, thread comments
- Profile and Organization settings
- Authenticate with Google, Github, and Username

### Technologies

- NextJS (v13)
- Zustand
- Typescript
- TailwindCSS
- Libs:
  - SwiperJS (v10)
  - ...
- Clerk, MongoDB
